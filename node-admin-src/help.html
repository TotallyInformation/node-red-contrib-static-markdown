Serves up a folder of Markdown files as a web site

Uses markdown-it with extensions to translate content to HTML. Uses Handlebars to wrap round the content.

Configuration

name string
Optional name shown in flow.
URL string
Required. The URL fragment appended to the httpNodeRoot (or / if that isn't set).
Source Folder string
Required. The filing system path to the root folder containing the Markdown source files to serve.
If the Node-RED setting httpStatic is set, that will be used as a default.
